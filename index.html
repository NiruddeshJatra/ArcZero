HOW TO PLAY

OBJECTIVE

Stop missiles before they hit the ground. Survive long enough, rack up enough intercepts, and clear the level.

CONTROLS

← → move launcher
↑ ↓ aim angle
Space hold to charge power, release to fire
Z flip launcher facing left/right
P or Esc pause · resume

SCORING

Every intercept above 20 m scores. Chain intercepts within 3 s to build a combo multiplier (up to ×8). High-altitude, clutch (low-altitude), and long-range hits earn extra score.

LEVEL COMPLETION

Three gates must all clear before a level ends — score threshold, minimum intercepts, and minimum waves. The level only advances during a RELEASE phase. The in-game pill under the HUD shows your progress.

MODES

Campaign — non-seeded random run from L1. Fully ranked.
Daily Challenge — all players get the same missile pattern today (seed changes each calendar day). One ranked attempt per day.
Level Select — mid-campaign start, marked unranked.

DAILY CHALLENGE — HOW

The seed is derived from today's ISO date (YYYY-MM-DD) via a hash — see rng.js seedFromDateISO. At local midnight a new date → new seed → new pattern. Spawns, angles, and missile kinds are deterministic per seed, so the same puzzle is replayable for fair comparison.
